# Fitness Gym - Система управления фитнес-центрами

Веб-приложение для управления сетью фитнес-центров, которое позволяет вести расписание, бронировать занятия, анализировать популярность тренировок и нагрузку тренеров.

## Функциональность системы

- Хранение информации о расписании, бронированиях, тренерах и клиентах
- Управление расписанием и бронированиями
- Пользовательский интерфейс для записи на занятия и управления бронированиями
- Автоматическое уведомление клиентов о предстоящих тренировках и изменениях расписания
- Аналитический модуль для формирования отчетов о популярности тренировок, загруженности тренеров и посещаемости
- Система отчетности с возможностью экспорта данных в популярные форматы (PDF, Excel)

## Технический стек

- **Backend**: Python, Flask
- **База данных**: SQLite
- **Frontend**: Flask Templates, Tailwind CSS
- **Уведомления**: Flask-Mail
- **Визуализация данных**: Chart.js
- **Экспорт отчетов**: ReportLab (PDF), XlsxWriter (Excel)

## Архитектура приложения

Приложение построено на архитектуре MVC (Model-View-Controller):

- **Model**: Модели данных для работы с SQLite
- **View**: HTML шаблоны с использованием Jinja2 и стилями Tailwind CSS
- **Controller**: Контроллеры маршрутов Flask, обрабатывающие запросы

## Типы пользователей

### Обычные пользователи

- Регистрация и авторизация
- Просмотр расписания тренировок
- Фильтрация тренировок по типам
- Бронирование тренировок
- Отмена бронирований
- Просмотр истории посещений

### Администраторы

- Весь функционал обычных пользователей.
- Управление тренерами (добавление, редактирование, удаление)
- Управление типами тренировок
- Управление расписанием тренировок
- Просмотр информации о пользователях
- Доступ к аналитике (популярность тренировок, загруженность тренеров, посещаемость)
- Создание отчетов в форматах PDF и Excel

## Установка и запуск

1. Клонируйте репозиторий
2. Создайте виртуальное окружение Python
3. Установите зависимости из файла `requirements.txt`:
```
pip install -r requirements.txt
```
4. Создайте файл `.env` на основе `.env.example` и заполните необходимые переменные окружения
5. Запустите приложение:
```
python app.py
```

## Конфигурация электронной почты

Для корректной работы уведомлений по электронной почте, необходимо настроить соответствующие переменные окружения в файле `.env`:

```
# Настройки SMTP-сервера
MAIL_SERVER=smtp.gmail.com
MAIL_PORT=587
MAIL_USE_TLS=True
MAIL_USERNAME=your_email@gmail.com
MAIL_PASSWORD=your_app_password
MAIL_DEFAULT_SENDER=no-reply@fitnessgym.com

# Для отключения отправки писем (в тестовой среде)
MAIL_SUPPRESS_SEND=False
```

Обратите внимание:
- При использовании Gmail необходимо создать пароль приложения (App Password) в настройках Google Account
- Для тестовой среды можно установить `MAIL_SUPPRESS_SEND=True`, чтобы отключить фактическую отправку писем
- Система обрабатывает ситуации с отсутствием настроек почты, поэтому отсутствие этих параметров не приведет к сбою приложения
