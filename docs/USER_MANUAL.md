## Введение

Fitness Gym - это веб-приложение для сети фитнес-центров, которое позволяет:
- Просматривать расписание тренировок
- Записываться на тренировки
- Отменять бронирования
- Получать уведомления о предстоящих тренировках и изменениях в расписании

Данное руководство содержит информацию о том, как пользоваться приложением для обычных пользователей.

## Регистрация и вход в систему

### Регистрация нового пользователя

1. На главной странице нажмите кнопку "Зарегистрироваться"
2. Заполните все обязательные поля формы:
   - Email (будет использоваться для входа и для получения уведомлений)
   - Имя
   - Фамилия
   - Пароль (минимум 6 символов)
   - Подтверждение пароля
3. Нажмите кнопку "Зарегистрироваться"
4. После успешной регистрации вы будете перенаправлены на страницу входа

### Вход в систему

1. На главной странице нажмите кнопку "Войти"
2. Введите ваш email и пароль
3. При желании, вы можете отметить опцию "Запомнить меня", чтобы не вводить данные при следующем посещении
4. Нажмите кнопку "Войти"
5. После успешного входа вы будете перенаправлены на главную страницу

### Выход из системы

1. Нажмите кнопку "Выйти" в верхнем меню
2. Вы будете автоматически перенаправлены на главную страницу

## Главная страница

На главной странице отображается:
- Приветственный баннер
- Информация о типах тренировок, доступных в фитнес-центре
- Список ближайших тренировок с возможностью быстрой записи

Для быстрой записи на тренировку с главной страницы:
1. Найдите интересующую вас тренировку в таблице "Ближайшие тренировки"
2. Нажмите кнопку "Записаться" напротив выбранной тренировки
3. Подтвердите запись на странице подтверждения

## Просмотр доступных тренировок

Для просмотра полного списка доступных тренировок:

1. Нажмите "Тренировки" в верхнем меню
2. На странице будут отображены все доступные тренировки

### Фильтрация тренировок

Для удобства поиска вы можете отфильтровать тренировки по типу:
1. На странице тренировок выберите интересующий вас тип из списка в верхней части страницы
2. Список тренировок будет автоматически отфильтрован

## Запись на тренировку

### Запись через страницу тренировок

1. Перейдите на страницу "Тренировки" через верхнее меню
2. Найдите интересующую вас тренировку в списке
3. Нажмите кнопку "Записаться" напротив выбранной тренировки
4. На странице подтверждения проверьте информацию о тренировке:
   - Дата и время
   - Тип тренировки
   - Тренер
   - Количество свободных мест
5. Нажмите кнопку "Записаться" для подтверждения
6. После успешной записи вы увидите сообщение о подтверждении и будете перенаправлены на страницу вашего расписания
7. На указанный вами при регистрации email будет отправлено уведомление с деталями записи

## Управление личным расписанием

Для просмотра и управления вашим расписанием:

1. Нажмите "Моё расписание" в верхнем меню
2. На странице будут отображены две секции:
   - "Предстоящие тренировки" - тренировки, на которые вы записаны
   - "История тренировок" - тренировки, которые уже прошли

### Просмотр деталей тренировки

На странице расписания для каждой тренировки отображается:
- Дата и время проведения
- Тип тренировки
- Тренер
- Для прошедших тренировок - статус посещения (посещено/не посещено)

## Отмена записи на тренировку

Для отмены записи на тренировку:

1. Перейдите на страницу "Моё расписание" через верхнее меню
2. В разделе "Предстоящие тренировки" найдите тренировку, запись на которую вы хотите отменить
3. Нажмите кнопку "Отменить" напротив выбранной тренировки
4. Подтвердите отмену в диалоговом окне
5. После успешной отмены вы увидите сообщение о подтверждении, и тренировка будет удалена из вашего расписания

> **Примечание:** Отменить можно только предстоящие тренировки. Отмена прошедших тренировок невозможна.

## Уведомления

Система автоматически отправляет уведомления на ваш email в следующих случаях:
- При записи на тренировку (подтверждение)
- За день до тренировки (напоминание)
- При изменении деталей тренировки администратором
- В случае отмены тренировки администратором
